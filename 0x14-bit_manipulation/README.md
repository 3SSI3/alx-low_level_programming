# Bit Manipulation
How to manipulate bits and use bitwise operators.

**Bitwise Operators** do bitwise manipulation eg. 
- AND operator; **&**.
- OR operator; **|**.
- NOT operator; **~**.
- Left shift; **<<**.
- Right shift; **>>**.
- XOR operator; **^**.
